## 📌 Présentation du Projet
**ASWAN** est une plateforme e-commerce multi-vendeur majeure opérant en Arabie Saoudite (KSA). Elle assure la liaison entre les marchands et les clients pour une gestion fluide des ventes et des livraisons.

### Mon Rôle & Responsabilités (Phylia Technology)
En tant que **Développeuse Flutter**, j'ai dirigé la création de toute la partie mobile :
* **Expertise UI** : J'ai réalisé l'intégralité de l'interface utilisateur (UI) avec l'équipe Flutter, garantissant une expérience fluide, moderne et bilingue (Arabe/Anglais).
* **Logique Métier** : Implémentation complète de la logique applicative via le pattern **BLoC**.
* **Intégration API** : Consommation exclusive des services REST fournis par l'équipe Backend Laravel via le client **Dio**.

---

## 🔍 Focus Technique & Description Étendue

#### 🏗️ Architecture Mobile & Leadership UI
* **Clean Architecture** : Utilisation stricte d'une structure en couches (Data, Domain, Presentation) pour assurer une séparation totale des responsabilités.
* **State Management** : Utilisation du pattern **BLoC** pour transformer les événements utilisateur en états réactifs et prédictifs, évitant les comportements instables.

#### 🚀 Optimisation & Consommation d'API
* **Performance** : Réduction du temps de chargement de l'interface utilisateur de **25%** grâce à une gestion d'état optimisée et un traitement efficace des flux de données.
* **Support RTL** : Adaptation complète de l'interface pour le marché saoudien, incluant le support natif du Right-to-Left (Arabe).

#### 🤝 Collaboration Technique
* **Synergie Backend** : Collaboration étroite avec les ingénieurs backend pour définir les contrats d'API et assurer une synchronisation parfaite des données.
* **Méthodologie Agile** : Participation active aux cycles de développement itératifs et aux revues de code pour maintenir des standards de qualité élevés.
